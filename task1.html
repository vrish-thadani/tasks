🌐 What is HTML?

HTML (HyperText Markup Language) structures content on the web. Tags like <h1>, <p>, and <a> define elements in a readable and organized format.

🧩 What are Semantic Tags?

Semantic HTML tags like <article> and <footer> clearly describe the role of content, improving SEO, accessibility, and code clarity.

🆔 ID vs Class in CSS
  • id: Unique identifier, used once per page (#id in CSS).
  • class: Reusable, used for styling multiple elements (.class in CSS).
  • Elements can have multiple classes, but only one id.
  • id has higher specificity than class.
📂 What is the <summary> Tag?

<summary> acts as a clickable heading for <details>, allowing collapsible sections like this one.

🎨 What is CSS?

CSS (Cascading Style Sheets) defines how HTML elements appear. Three ways to apply:

  • Inline: via style="" attribute
  • Internal: inside <style> tags
  • External: linked .css file via <link>
